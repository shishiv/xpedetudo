# 🍔 xpedetudo

**xpedetudo** is an open-source automation tool for restaurants and food trucks.  
It handles orders, inventory, shopping lists, and recipe management.

The platform features:
- 📊 A web interface for the business owner to register products, monitor usage, and generate reports like:
  - “What did I use today?”
  - “What do I need to buy for tomorrow?”
- 💬 A WhatsApp interface for customers to place orders and interact naturally.
- 🧠 Integration with OpenAI to:
  - Generate shopping lists from usage trends
  - Suggest inventory optimization
  - Auto-describe products or recipes
  - Answer owner questions like “how much did I sell of X last week?”

## ✨ Tech Stack

- Supabase (Database & Auth)
- Node-RED & n8n (Process Automation)
- Grafana (Reports & Monitoring)
- OpenAI Codex / GPT (AI features)
- WhatsApp Business API
- React (Web Dashboard)
